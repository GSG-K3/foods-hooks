## How to run the project locally

- Clone this repo to your local machine using terminal:
```shell
git clone https://github.com/GSG-K3/foods-hooks.git
```
- run these commands in your terminal:
```shell
 $ cd foods-hooks
 $ npm i
 $ npm start
```
